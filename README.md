# LevelChartPlugin

艦娘 Level チャート を表示する KanColleViewer Plugin です。

![Sample](LevelChartSample.png)

![Sample](LevelTableSample.png)


### インストール

* `LvChartPlugin.dll` を KanColleViewer の `Plugins` ディレクトリに放り込んで下さい。
